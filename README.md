# Meteor Dodger (メテオ・ドジャー)

宇宙船を操り、降り注ぐ隕石を回避し続けるサバイバルアクションゲームです。Antigravityを使用。

## 遊び方
### 操作方法
- **移動**: 矢印キー (`←` `→`) または `A` `D` キー
- **リスタート**: ゲームオーバー時に `Space` または `Enter`

### ルール
- 隕石に当たると即ゲームオーバーです。
- 時間経過とともにスコア（Base Score）が加算されます。
- アイテムを活用して高得点を目指しましょう。

### アイテム
- **★ (スター)**: 5秒間「無敵」になります。無敵中に隕石に当たると破壊してボーナス点(+100pts)が入ります。
- **TIME (時計)**: 5秒間「減速」します。隕石の落下スピードが半分になります。

### 難易度
- **Easy / Normal / Hard**: 最初から一定の難易度で遊びたい人向け。
- **Progressive**: 時間経過とともに徐々に難しくなるモード。
